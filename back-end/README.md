To set up the back end, follow these steps:

    -   First clone this repository using whichever cloning option works best

    -   Then, type the following into the terminal: "python -m venv"

    -   After which, type "source .venv/Scripts/activate" in bash terminal

        -   Currently, the terminal should have a "(.venv)" above the header

    -   Finally, type "pip install -r requirement.txt"
