To set up the back end, follow these steps:

    -   First clone this repository using whichever cloning option works best

    -   Then, type the following into the terminal: "python -m venv [name-of-virtual-environment]"

    -   After which, type "source .[name-of-virtual-environment]/Scripts/activate" in bash terminal

        -   Currently, the terminal should have a "(.[name-of-virtual-environment])" above the header

    -   If you're using powershell, type "& .[name-of-virtual-environment]/Scripts/activate.ps1"

        -   Currently, the terminal should have a "(.[name-of-virtual-environment])" to the left of the header

    -   Finally, type "pip install -r requirement.txt" and all packages and libraries should be installed
